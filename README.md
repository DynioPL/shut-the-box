# shut-the-box

### A stupidly simple implementation of popular game of dice.

This project **IS NOT** complete. It will be updated.

## Building

### *nix (Linux, BSD...)
`shut-the-box` comes with a Makefile for *nix userspace.
`make`

### Windows
Either use some *nix toolchain (Cygwin, MSYS2...), or compile it manually.
(eg. for MSVC)
`cl /o shut-the-box '.\src\main.c'`
